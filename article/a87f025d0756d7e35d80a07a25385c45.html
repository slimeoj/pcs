题目大意 && 解析

有NNN个点,每个点可以染成444种颜色,每个点关联的点不超过333个(也就是说一定有解!),关联点颜色不能一样,答案字典序最小.

思考:

第一个点,一定就111,因为一定有解,无论111号位置上的值就多少,利用替换的方式,都可以变成111

第二个点,一定就111或222,按同样的思想来.

如果1,21,21,2有冲突,那222号可以就2,3,42,3,42,3,4,同样可以用替换的思想变成222

代码

copy
#include <iostream>
#include <fstream>
using namespace std;

int main(void)
{
    int N, M;
    int A[151], B[151], G[101];
    ifstream fin ("revegetate.in");
    fin >> N >> M;
    for (int i=0; i<M; i++) { 
        fin >> A[i] >> B[i];
        if (A[i] > B[i]) swap (A[i], B[i]);
    }

    ofstream fout ("revegetate.out");  
    for (int i=1; i<=N; i++) {
        int g;
        for (g = 1; g <= 4; g++) {
            bool ok = true;
            for (int j=0; j<M; j++) 
	            if (B[j] == i && G[A[j]] == g) ok = false;
            if (ok) break;
        }
        G[i] = g;
        fout << g;
    }
    fout << "\n";
    return 0;
}
